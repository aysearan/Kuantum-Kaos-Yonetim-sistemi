# Kuantum-Kaos-Yonetim-sistemi
Bu proje nesneye yönelik progamlama konularının farklı dillerde kullanımını göstermek amacıyla tasarlanmıştır. Projede kalıtım, interface, abstract class, class, kapsülleme ve custom exception kullanımına yer verilmiştir. Proje c# , python, java, javascript dillerinde ayrı ayrı yazılmıştır. Ve bu dillerin kullanım farklılıkları gözlemlenir.

 Programın temel işleyişi; kullanıcıya bir menü sunmak, seçimlere göre nesne eklemek, listelemek, analiz yapmak ve hata durumlarını özel mesajlarla göstermek üzerine kuruludur. Ve bunu hikayeleştirerek eğlenceli bir deneyim sunar. 

 Çalıştırma şekilleri:

 java için terminale aşağıdaki kod yazılmalıdır:
 node App.js

 Diğer diller için(c#,python,java):
 Sağ üstte bulunan üçgen çalıştırma butonuna basmak yeterlidir.
